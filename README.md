# Where is this place?

[Where is this place](https://whereisthisplace.net/) is an AI-powered photo geolocation tool that helps you answer one question: **“Where was this photo taken?”**

Instead of guessing based on vague visual hints, you can upload an image and let the system analyze it using a mix of computer vision and metadata signals.

## What this website does

- **AI photo locator** – Analyzes skylines, buildings, landscapes, and other visual clues to suggest likely locations.
- **EXIF & metadata scan** – When available, reads GPS coordinates, timestamps, and camera information embedded in the file.
- **Landmark recognition** – Detects well-known monuments, storefronts, and natural features to narrow down the map.
- **Map-based investigation** – Shows candidate coordinates you can review and validate yourself.

## Who it is for

- **Travelers** who want to rediscover where old trip photos were taken.
- **Creators** who need accurate location context for content, captions, or storytelling.
- **OSINT and investigators** doing open-source research on where images originated.

## How to use

1. Visit **https://whereisthisplace.net/**.
2. Upload a photo from your device.
3. Wait for the AI to analyze visual features and metadata.
4. Review the suggested locations and supporting evidence.

Where Is This Place turns a single image into a starting point for deeper geographic investigation.
